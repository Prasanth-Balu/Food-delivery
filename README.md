#  Food Delivery Platform

A Full Stack Food Delivery web application built using the MERN Stack.
The application allows users to browse food items, add products to the cart, place orders, and make secure online payments.
It also includes a dedicated Admin Panel for managing food items and customer orders.


# Features

## User

- User Registration
- User Login
- JWT Authentication
- Browse Food Menu
- View Food Details
- Add to Cart
- Remove Items from Cart
- Place Orders
- Online Payment using Stripe

## Admin

- Add Food Items
- Upload Food Images
- Update Food Details
- Delete Food Items
- View Customer Orders
- Manage Order Status

# Installation

## Backend

npm run server

## Frontend

npm install
npm run dev


## Admin

cd admin
npm install
npm run dev
